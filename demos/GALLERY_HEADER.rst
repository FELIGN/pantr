Demo gallery
============

Standalone, runnable scripts illustrating the main features of PaNTr. Each
example below is generated by executing the matching script in ``demos/``; the
same scripts run on their own::

    pip install "pantr[viz]"          # the 3-D demos need the viz extra
    python demos/01_bspline_geometry_tour.py

The 3-D scenes are interactive — drag to rotate, scroll to zoom. The plotting
demos use Numba kernels; on a fresh process you may hit a Numba
background-warmup threading error, in which case run with JIT disabled (the
documentation build does this automatically)::

    NUMBA_DISABLE_JIT=1 python demos/05_approximation.py

The examples, in order:

- **01 — B-spline geometry tour**: curves/surfaces, derivatives, Greville
  points, an exact NURBS circle (:mod:`pantr.bspline`, :mod:`pantr.cad`).
- **02 — Visualization basics**: ``plot`` / ``Scene``, control polygon, knot
  lines, scalar fields, VTK export (:mod:`pantr.viz`).
- **03 — Basis gallery**: Bernstein / Lagrange / Legendre bases and a
  change-of-basis matrix (:mod:`pantr.basis`, :mod:`pantr.change_basis`).
- **04 — Knot operations**: knot insertion, degree elevation, Bézier extraction
  (:mod:`pantr.bspline`).
- **05 — Approximation**: interpolation / L2 projection / quasi-interpolation
  and convergence (:mod:`pantr.bspline`).
- **06 — CAD modeling**: primitives plus extrude / revolve / ruled, assembled
  (:mod:`pantr.cad`, :mod:`pantr.transform`).
- **07 — Bézier and roots**: a Bézier surface, Bernstein root finding, and a
  curve–line intersection (:mod:`pantr.bezier`).
- **08 — THB adaptive refinement**: local refinement with the hierarchical mesh
  and per-level control net (:mod:`pantr.bspline`, :mod:`pantr.grid`).
- **09 — Grids and quadrature**: grids, ``cell_quadrature`` integration, a BVH
  query, grid rendering (:mod:`pantr.grid`, :mod:`pantr.quad`,
  :mod:`pantr.geometry`).
- **10 — Transforms**: affine translation / rotation / scaling / shear and
  composition (:mod:`pantr.transform`).
